hexo的安装（win版）

  • 安装准备

    • 安装git
    • 安装Node.js

  • 下载hexo

    打开git bash 输入npm install -g hexo-cli即可。可通过hexo -v查看版本

  • 创建hexo的所需文件

    在自己的所创文件下打开git bash ,输入hexo init即可。

    其中\source_posts文件夹下是放用户写的的博客md文件的,用hexo需用hexo new 文件名来创建md，不能自己直接创建(会在页面显示上没有md文件的名称),在\themes下是放hexo的主题的，目前只有landscape主题，可去hexo 主题下载，本人较为喜欢next主题（嘿嘿）

    • 他的配置文件说明可看官网,其中git上的配置官网上都有，若懒得看官网就看我下面的说明。
    • 写好你的文章后，在同一文件下打开git bash,输入hexo g生产html文件，再hexo s会在本地生产服务器，注上传到git上时不能省略hexo s这一步。

  • 部署到git上

    • 在_config.xml上的末尾添加这些：注意github page要的分支为master或没有gh-pages，且git地址要是用户名.github.io其中.github.io是创建github page的要求我其他的试了不行反正是git上的也就不管了，否则站点无法发布。具体看官网说明

    • 在 git上创建github page(我按我的部署来简要说明)

      先create 一个Repository，然后create brash 为gh-pages,然后点击Settings，下面有一个GitHub Pages，其中Source下可以选择发布的站点，选择后保存即可，会出现Your site is ready to be published at https://xxxxx.。https://xxxxx即为访问地址。

    • 然后在git bash 上输入hexo d会将生成的html等文件上传到guthub 上，到此博客建立完成。
